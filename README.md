# Heavenly Birthday Blessings

A peaceful Christian birthday page with angels, scripture, and gentle golden light.

## Open locally

```bash
cd HappyBirthday
python3 -m http.server 8080
```

Visit [http://localhost:8080](http://localhost:8080), or open `index.html` in your browser.

## Online

Upload the folder (or just `index.html`, `styles.css`, and `app.js`) to [Netlify Drop](https://app.netlify.com/drop) for a free shareable link.

## Customize

- Type a name in the field on the page for a personal greeting.
- Click **Receive a Birthday Blessing** for wishes, soft chimes, and a glow of light.
